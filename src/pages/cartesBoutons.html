Boutons customisés :

Pour insérer un effet de bouton, il suffit d'utiliser la classe associée à celui-ci. Les classes sont indiquées en titre des cartes.

Pour utiliser les classes de boutons qui suivent, il est INDISPENSABLE d'ajouter une classe de palette dans le HTML. Par exemple : primary btn-1-full. Cette classe correspond à la couleur de palette associée ($color-palette).

La classe .button (classe de base des boutons dans Zurb Foundation) doit être ignorée lorsqu'on ajout une classe issue de la bibliothèque ci-dessous.

Remarque : Les classes suivantes peuvent avoir un comportement inattendu lorsque la classe .button de Zurb Foundation est insérée sur la même balise HTML.

* $button-color ou $button-color-alt : La couleur du texte est générée automatiquement pour optimiser le contraste entre le fond et le texte.

*** Si la classe $button-border-color-hover est paramétrée sur "none" ou "transparent", alors le bouton prend pour couleur de police et bordure la couleur paramétrée dans la variable $button-color-hover.

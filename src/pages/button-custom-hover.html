Boutons customisables :

Insérer des couleurs dans la palette. (Customization sass / 1. General / $foundation-palette )

Les classes sont indiquées en haut des cartes. Il est possible de combiner toutes les classes (sauf indication contraire mentionnée sur la carte).

Pour utiliser les classes de boutons qui suivent, il est INDISPENSABLE d'ajouter une classe de palette dans le HTML. Par exemple : primary btn-full. Cette classe correspond à la couleur de palette associée ($color-palette).

La classe .button (classe de base des boutons dans Zurb Foundation) doit être ignorée lorsqu'on ajout une classe issue de la bibliothèque ci-dessous. Les classes qui suivent peuvent avoir un comportement innatendu avec l'insertion de la classe .button.
